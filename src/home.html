How to Navigate Solar System
  1. To zoom in put the mouse where you want to zoom and click.
  2. To zoom out hold shift and click where you want to zoom out.
  3. To pan you have to click, hold, drag to where you want to pan, and release when you are done.
  4. If you get lost hit the reset button.
Formulas:
  • planet_radius_in_pixels = sun_radius_in_pixels * (planet_real_radius_in_km / sun_real_radius_in_km)

  1. I researched how to make a scaled down model of the solar system using JavaScript and HTML5.
    This is the blog I used to start my project from.
  2. I cloned the open source project from the blog as a starting point for my project.

    Notes: The original project did not have:
    • The planets size scaled relative to the sun.
    • The planet distance scaled relative to the sun.
    • The planet orbit velocity relative to other planets.
    • The ability to zoom in and out of the solar system model.
    • The ability to pan around the solar system model.
  3. With my mom and dads help, I added the code to scale objects and distances, zoom in and out, pan around, and host the project on the Cloud.

The planets in the model are scaled relative to the size of the Sun. In order to fit all eight orbits of the planets, I had to set the size of the canvas to 13,000 X 13,000 pixels when the Suns radius is set to 50 pixels in the model. This scale is made clear in the image on the right.

Note: Please follow the instructions to zoom in to see the planets close up.
