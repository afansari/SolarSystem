How to Navigate Solar System
  1. To zoom in put the mouse where you want to zoom and click.
  2. To zoom out hold shift and click where you want to zoom out.
  3. to pan you have to click, hold, drag to where you want to pan, and release when done.
  4. If you get lost hit reset

  1. I researched how to make a scaled down model of the solar system using JavaScript and HTML5.
    This is the blog I used to start my project from.
  2. I cloned the open source project from the blog as a starting point for my project.

    Notes: The original project did not have:
    • The planets size scaled relative to the sun.
    • The planet distance scaled relative to the sun.
    • The planet orbit velocity relative to other planets.
    • The ability to zoom in and out of the solar system model.
    • The ability to pan around the solar system model.
